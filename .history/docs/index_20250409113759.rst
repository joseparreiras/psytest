psytest documentation
=====================

This is the documentation for the psytest package, a Python testing framework that applies the methodology of :cite:`phillips2011testing` (PSY) to test for the presence of multiple bubbles in a dataset. 

A bubble is defined as a period of explosive growth above a unit root in the time series. The PSY test applies a sequence of backward expanding Augmented-Dickey Fuller (ADF) tests to the data in order to identify the presence of bubbles. The benefits of this test relative to other tests in the literature is its ability to identify multiple bubbles in a dataset.
